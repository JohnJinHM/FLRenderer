> **English** | [中文](#中文说明)

A web-based tool designed specifically for history, geography, and military content creators. It allows you to easily create animated frontline maps, visualize territorial changes, and export them as high-quality video materials directly from your browser—no complex After Effects / Premiere Pro masking required.

## ✨ Features

* **Custom Base Maps:** Import any high-resolution map image from your local device.
* **Smart Spline Drawing:** Simply click to plot nodes; the tool automatically generates smooth, glowing frontline curves.
* **Keyframe Animation:** Move the playhead, drag the nodes, and let the timeline automatically interpolate the smooth expansion or retreat of territories.
* **Full Customization:** Create multiple tracks with customizable colors, outline widths, and opacity to represent different factions (e.g., Red Team vs. Blue Team).

> [English](#frontline-animator--战线动态生成器) | **中文**

一款为历史、地理和军事类视频创作者打造的网页端辅助工具。告别在 AE 或 PR 中繁琐的遮罩抠图，在浏览器中即可轻松制作战线推移、领土变更的动态地图，并导出为高质量视频素材。

## ✨ 功能特性

* **自定义底图：** 支持随时导入任意本地高清地图图片作为背景。
* **智能平滑绘制：** 仅需通过鼠标点击生成关键点，算法会自动将其连接为平滑、带发光效果的战线边缘，无需手动调节复杂的贝塞尔手柄。
* **关键帧动画引擎：** 拖动时间轴并移动地图上的节点，系统会自动生成关键帧并计算平滑的推移过渡动画。
* **全面自定义：** 支持创建多个独立轨道，自定义不同阵营的颜色、描边粗细和填充透明度（例如红蓝对抗）。
